<div align="center"><sub>02 &nbsp;——&nbsp; FEATURED PROJECTS</sub></div>

<br/>

**`01` &nbsp; RailSeat — Full-Stack Railway Reservation Platform**

Built a high-concurrency reservation engine using **React, Node.js, and PostgreSQL**. Implemented real-time seat synchronization via **Socket.io** to prevent race conditions at checkout. Engineered an interactive dynamic coach visualization grid.

![React](https://img.shields.io/badge/React-0d1117?style=flat-square&logo=react&logoColor=3399ff&labelColor=161b22)

<br/>

**`02` &nbsp; Vision Guard — Privacy-First Computer Vision Tool**

Local, hardware-accelerated Python utility using the **YOLO** object detection model. Automatically triggers desktop screen-blur on detecting mobile devices in-frame — protecting sensitive source code and on-screen data in real time.

![Python](https://img.shields.io/badge/Python-0d1117?style=flat-square&logo=python&logoColor=3399ff&labelColor=161b22)
![YOLO](https://img.shields.io/badge/YOLO-0d1117?style=flat-square&logo=yolo&logoColor=3399ff&labelColor=161b22)
![OpenCV](https://img.shields.io/badge/OpenCV-0d1117?style=flat-square&logo=opencv&logoColor=3399ff&labelColor=161b22)

<br/>

**`03` &nbsp; Multi-Agent Routing Engine & Core Networks**

Decentralized multi-agent system for smart routing path optimization. Custom reliability, sliding-window flow control, and packet acknowledgment layers implemented natively in **C** over stateless UDP connections.
